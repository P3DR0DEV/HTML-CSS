Paragrafos e quebras de linhas

Paragrafos podem ser escritos de qualquer forma, sempre serão em sequências. Para quebrar um paragrafo, utilizar a Tag <br>

Para representar os simbolos de maior que > e menor que < em uma sequencia de comando: < e >.

| Cent: ¢ |

Adicionar Mídias

Imagens

Para adicionar imagens, é bem simples, basta adicionar a tag <img> com o scr="Destino onde a imagem se encontra" e alt="Sobre o que a Imagem se refere.
Exemplo: "

Imagens expansíveis

Para adicionar imagens expansíveis, e o seu site ser acessível em diferentes dispositivols sem quebrar a formação, utilizamos a tag <picture> , dentro da indentação de tag picture, entra a tag <img> como imagem principal, e a tag <source> buscará a imagem de tamanho alternativo.

Áudios

Para adicionar audios, usamos a tag <audio> e dentro dessa tag, utilizamos a tag >source:src< com os parametros (preload); (controls); (loop [caso queira um áudio em looping])

Adicionar Vídeos

Para adicionar vídeo hospedados no próprio computador, utilizamos a tag <video> e <source:src>

Adicionar Ícone

Para adicionar uma logo para icone devemos usar o formato de imagem "ICO", para pegar um icon pronto basta buscar "icon archive" no Navegador, para transformar uma imagem jpg, png em icon, basta utilizar o favicon.io! Para colocá-lo, basta usar a tag <Link:favicon> na área Head do html.
